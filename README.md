# election-ai-pro
Election Mitra - AI Election Guide 🇮🇳  An interactive AI assistant that helps Indian citizens understand the election process, voting steps, and timelines. Built for PromptWars Challenge 2.  Features: - 🤖 Gemini AI-powered chat assistant - 🗳️ Voter registration guidance - 📟 EVM & VVPAT explained - 📋 Voting day checklist - 📍 Polling booth finder - 🏛️ Lok Sabha election process - 📊 Real election data (2024)  Tech Stack: - Backend: Flask (Python) - AI: Google Gemini 2.0 Flash - Frontend: HTML, CSS, JavaScript - Data: JSON with real Indian election data  Built by Mohd Waseeq Siddiqui
